HTTP

معلومات الزائر والمعلومات الاخرى

ماهو HTTP

يعتبر الكائن http هو كائن مسئول عن عرض وجلب وتخزين بيانات Hypertext Transfer Protocol , حيث يمكنك معرفة المزيد عن HTTP عبر زيارة ويكيبيديا من هنا, من مميزات هذا الكائن معرفة بعض المعلومات الاساسية عن المستخدم او زائر موقعك كمثال لا الحصر ( عنوان IP - معلومات المتصفح - نوع نظام التشغيل ) .. ألخ ويمكنك معرفة المزيد عن هذا الكائن ادناه .


استخدام الكائن

لاستخدام الكائن http يمكنك طلبه بشكل ادناه :

$this->http;


معلومات الزائر ( المستخدم )

لمعرفة بيانات الزائر يمكنك طلب المتغيير user , وهو متغيير يقوم بحفظ بيانات الزائر بشكل تلقائي بمجرد دخوله إلى الموقع الالكتروني الخاص بك, ان المتغيير user هو متغيير ذو صفة مصفوفة مفهرسة index array يحتوي على الفهارس ادناه :

  • browser اسم المتصفح
  • platform اسم نظام التشغيل
  • robot اسم الروبوت كمثال GoogleBot وتكون قيمة هذه الفهرسة خالية في حال لم روبوت او عنكبوت
  • is_robot وهي فهرسة تأخذ قيمة true ان كان روبوت او عنكبوت آلي false ان لم يكن
  • mobile وهي فهرسة خالية الا ان كان الزائر يتصفح الموقع من جهاز متنقل ( الاجهزة الذكية مثلاً كأيفون - ايباد - جلاكسي .. ألخ )
  • is_mobile وهي فهرسة تأخذ قيمة true ان كان الزائر يتصفح من جهاز متنقل و false ان لم يكن
  • version وهي فهرسة خاصة لرقم اصدار المتصفح
  • language وهي اللغات المستخدمة والمدعومة بالمتصفح
  • agent وهو بيانات العميل ( الزائر ) كاملة كنص واحد
  • ip وهو عنوان IP ويمكنك معرفة المزيد حول IP عبر زيارة ويكيبيديا من هنا

حتى تتمكن من جلب هذه البيانات يمكنك عرضها عن طريقة وظيفة print_r او الوظيفة الخاصة بالمنصة pre_print_r كمثال ادناه :

print_r($this->http->user);

ولطلب معين كمثال لنوع المتصفح يمكنك الطلب كأدناه :

echo $this->http->user['browser'];

وهناك طرق كثيرة للاستخدامات اعلاه فمثلاً لو اردت ان تتحقق من ان المستخدم هل هو عنكبوت او روبوت وتريد وضع شروط له يمكنك ذلك كالمثال ادناه لا الحصر :

if ($this->http->user['is_robot'] == true) {
    echo "Its a robot from ".$this->http->user['robot'];
} else {
    // Do somthing
}


اضافة\تعديل في البيانات

يمكنك الذهاب إلى المسار application/config/http ستجد أربع ملفات بها مصفوفات خاصة للمعلومات يمكنك اضافة او تعديل او حذف اي معلومات, الملفات ادناه مشروحه :

  • browsers.php مخصص لاسماء المتصفحات
  • mobiles.php مخصص لاسماء الاجهزة المتنقلة والذكية
  • robots.php مخصص لاسماء العناكب والروبوتات
  • platforms.php مخصص لاسماء المنصات وانظمة التشغيل

أن الكائن http سيتم تطويره بالمستقبل ليكون اكثر مرونة ذو مميزات اكبر مستقبلاً لكن مادون اعلاه هو اساس ثابت به ماسيتم اضافته لا يعني حذف ماذكر اعلاه .
